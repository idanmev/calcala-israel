# מדריך עיתונות פיננסית ישראלית — Hebrew Financial Journalist Skill File

> תוצאה של ניתוח מעמיק של כתבות מכלכליסט וביזפורטל (אפריל 2026).
> כל הדוגמאות אמיתיות, שאובות ממאמרים שנקראו ישירות.

---

## 1. כותרות — Headlines

### עקרונות
- **מבנה דו-חלקי עם נקודותיים**: "X: Y" — הראשון מציג את הגוף/אירוע, השני את הפרט הבולט
- **מתח וניגוד**: "אבל", "למרות", "רק" — מוסיפים מתח להצהרה
- **מספרים**: מספרים ספציפיים בכותרת מעלים קריאות מיידית
- **שאלה רטורית**: שאלה ישירה מושכת את הקורא לפנים

### דוגמאות אמיתיות (ממקורות שנסרקו)
```
✅ תרחיש קריסת אינטל ירד מהשולחן, אבל עדיין מוקדם לחגוג מפנה אמיתי
✅ גוגל מהמרת על קלוד: תשקיע באנתרופיק עד 40 מיליארד דולר
✅ 28 אלף שקל על יעוץ משכנתא? לקוחה התחרטה - היועצת אמרה "מה פתאום"
✅ אושר הסדר הפשרה בשטראוס: תקבל פיצוי של 27 מיליון שקל מחברת הביטוח כלל
✅ הברקס של שוק הנדל"ן לא התגלגל למשכנתאות
✅ 55% מהמקצועות עומדים להשתנות דרמטית: האם המשרה שלכם בטוחה?
✅ ההשתעבדות למשכנתא הגיעה ב-2025 לשיא חדש
✅ מכה לפאפאיה גיימינג: חבר מושבעים בניו יורק פסק כי תשלם 420 מיליון דולר
✅ הנגיד: "ריבית של 3.75% באזור ספטמבר 2026 נראית לי סבירה"
✅ גיוס ענק לדקארט הישראלית: 100 מיליון דולר לפי שווי של 3.1 מיליארד
```

### תבניות כותרת מנצחות
| תבנית | דוגמה |
|---|---|
| `[גוף] + [מהלך]: [מספר]` | גוגל מהמרת על קלוד: תשקיע עד 40 מיליארד |
| `[תרחיש]: [מה קרה בפועל]` | תרחיש קריסת אינטל ירד מהשולחן, אבל... |
| `[מחיר]? + [קונפליקט]` | 28 אלף שקל על יעוץ משכנתא? לקוחה התחרטה |
| `[אירוע משפטי]: [נתון]` | אושר הסדר הפשרה: תקבל 27 מיליון שקל |
| `[מספר] + [קטגוריה]` | 55% מהמקצועות עומדים להשתנות |

## 2. פסקת פתיחה — Opening Paragraphs

### חוקים
- **לעולם לא מתחילים ברקע**. הקורא לא רוצה הקשר לפני הבשר.
- **העובדה הכי מעניינת קודמת**.
- **3 סגנונות פתיחה עיקריים**: שאלה רטורית / הצהרה + ייחוס / תיאור סצנה קצר

### דוגמאות אמיתיות
```
1. שאלה רטורית (כלכליסט):
"האם החגיגה במניית אינטל מייצגת מפנה אמיתי, או שמדובר רק באנחת רווחה לאור
התרחקות תרחיש הקריסה עבור מי שהיתה בעבר מלכת שוק השבבים. ביום חמישי האחרון
פירסמה אינטל תוצאות כספיות לרבעון הראשון שהיו טובות מהצפוי, מה שהקפיץ את
המניה ב-24%."

2. הצהרה + ייחוס (כלכליסט):
"גוגל מתכננת להשקיע עד 40 מיליארד דולר בסטארט-אפ הבינה המלאכותית אנתרופיק –
כך הודיעו החברות היום (ו'), במהלך שממחיש את ההסלמה במירוץ הגלובלי לשליטה
בשוק ה-AI."

3. מי עשה מה (כלכליסט — חדשות משפטיות):
"שופט בית משפט המחוזי בתל אביב אריאל צימרמן אישר את הסדר הפשרה שנחתם בין
קבוצת שטראוס לבין חברת הביטוח של נושאי המשרה, כלל ביטוח."

4. משפט קצר ×2 — פאנץ' (ביזפורטל):
"יום אחד. זה כל מה שנדרש לליטל זוהר כדי להבין שעשתה טעות. יממה לאחר
שחתמה על חוזה עם חברת הייעוץ הפיננסי פנתה וביקשה לבטל."

5. תיאור סצנה (ביזפורטל — בלי מספרים):
"ביתם היה קר בחורף - ולא הודלק חימום. המקרר היה מלא בספרי קודש במקום אוכל.
הילדים לבשו בגדים מרופטים שעברו מיד ליד. כל אותו הזמן, לאב היו בחשבונות
בנק בארצות הברית למעלה מ-2 מיליון דולר."
```

## 3. קצב משפטים — Sentence Rhythm

### הכלל
**לא אחיד — זה הסוד.** שלב בין משפטים קצרים מאוד (4–7 מילים) למשפטים ארוכים (15–25 מילים). הקצב הזה מונע שעמום.

### דפוס מוצלח
```
[משפט ארוך — 15-20 מילים, עם עובדה מרכזית]
[משפט קצר — 4-8 מילים, מסכם או מנגיד]
[משפט ארוך — הרחבה]
```

### דוגמה מהשטח (כלכליסט):
```
"כעת המניה החבוטה הפכה לאחת הטובות בוול סטריט, לאחר שהכפילה את ערכה
מתחילת השנה. [ארוך]
אבל גם אחרי כל הסופרלטיבים, היא נסחרת בשווי של 414 מיליארד דולר. [בינוני]
הרחק מהמתחרות. [קצר מאוד — 3 מילים]"
```

### אסור להתחיל פסקה עם:
- "בנוסף לכך" / "יתרה מזאת" / "לסיכום" / "כמו כן" / "חשוב לציין"
- "על פי" (כשאין שם ספציפי שאחריו)
- "יש לציין כי" / "יצוין כי"

## 4. כותרות משנה — Subheadings

### שני סגנונות עיקריים

**סגנון כלכליסט — נרטיב/שם/זווית:**
```
✅ עם קצת עזרה מאלון מאסק
✅ מקום מרכזי גם לישראל
✅ הקרב... מתחמם
```
→ מספרים לרוב לא, מוסיפים זווית חדשה.

**סגנון ביזפורטל — נתון/ציטוט/שלב:**
```
✅ 7 עד 9 מיליון שקלים           [נתון גולמי כותרת]
✅ "מקרה ייחודי"                  [ציטוט מהשופט]
✅ "מצג שווא גמור"                [ציטוט דרמטי]
✅ 200 שקלים לילד                [נתון מזעזע]
✅ שימוש בלתי חוקי במסמכים       [תיאור עובדתי חד]
```

### כלל: כותרת משנה טובה עונה על "מה קורה כאן בדיוק?" תוך 5 מילים.

## 5. ניסוח מספרים — Number Formatting

### כללים
```
אחוזים:          ב-24%  /  ב-7%  /  ב-22%
                 (מקף לפני %, לא רווח)

מיליארדים:       13.6 מיליארד דולר  /  100 מיליון דולר
                 (ספרות + מיליארד/מיליון, לא מיליארד')

שקלים גדולים:   27 מיליון שקל  /  293 מיליון שקל
                 (לא ₪ בגוף הכתבה — ₪ רק בטבלאות/כותרות קצרות)

שקלים קטנים:    28,000 שקלים  /  34,000 שקלים
                 (מספר + "שקלים" — לא ₪34,000)

דולרים:          40 מיליארד דולר  /  10 מיליון דולר
                 (לא $40B — תמיד בעברית)

תאריכים:        ב-2022  /  ב-2025
                (ביום חמישי האחרון  /  היום (ו'))

שווי:            לפי שווי של כ-380 מיליארד דולר
                 (ה"כ-" לפני הסכום מציין הערכה)
```

### ייחוס יום הפרסום (ישראלי קלאסי)
```
"כך הודיעו החברות היום (ו')" — יום שישי
"כך נמסר הבוקר (ג')" — יום שלישי
```
→ ציין יום בסוגריים כשמדובר בידיעה שוטפת.

## 6. מילים לשימוש ומילים להימנע — Vocabulary

### ✅ השתמש (אותנטי לעיתונות ישראלית)
| מילה/ביטוי | הקשר |
|---|---|
| אנחת רווחה | תגובה לחדשות חיוביות אחרי חשש |

| ענקית ה- | ענקית הטכנולוגיה, ענקית השבבים |
| הסלמה | escalation בתחרות/מחירים |
| זינק / זינקה | עלה בחדות (מניה, מחיר) |
| הקפיץ | caused to jump/spike |
| שוברת שיאים | record-breaking |
| קאמבק | comeback (מקובל בעברית ישראלית) |
| אקזיט | exit (IPO or acquisition) |
| גיוס ענק | large fundraise round |
| שווי / שווי חברה | valuation |
| תמהיל | mix (mortgage basket) |
| גורם מלווה | lending institution |
| נושאי המשרה | company officers/executives |
| ניגוד עניינים | conflict of interest |
| ועדה בלתי תלויה | independent committee |
| הסדר פשרה | settlement agreement |
| תביעה נגזרת | derivative lawsuit |
| שכר טרחה | professional/attorney fees |
| עסקה מתמשכת | ongoing transaction (consumer law) |
| חוק הגנת הצרכן | Consumer Protection Law |

| בנוסף לכך | בנוסף |
| יתרה מזאת | [פתח פסקה חדשה] |

## 7. סגנון סיום — Closing Paragraphs

### שלושה סגנונות מוצלחים

**1. הבטה קדימה עם הסתייגות (כלכליסט — אנליזה):**
```
"עד אז, ההתנפלות על מניית אינטל יכולה להיות מוסברת גם בכך שהמשקיעים מחפשים
את ההזדמנויות הזולות יותר, לאחר שחברות כמו אנבידיה או ברודקום כבר מתומחרות
באופן מלא."
→ משאיר שאלה פתוחה בראש הקורא.
```

**2. הרחבה ישראלית — "מה זה אומר עלינו" (כלכליסט):**
```
"בהקשר של AI חשוב לציין את המקום המרכזי של הפעילות הישראלית. כך למשל,
קרין אייבשיץ סגל היא לא רק מנכ"לית אינטל ישראל, אלא גם מובילה את האחריות
ההנדסית על חטיבת ה-Data Center Group של החברה כולה."
→ מחבר את הסיפור הגלובלי לישראל.
```

**3. פסיקה/עובדה אחרונה (ביזפורטל — מקרים משפטיים):**
```
"הנתבע תשלם לזוהר 34,000 שקלים בתוך 30 יום, ובנוסף 1,500 שקלים הוצאות משפט."
→ נתון קונקרטי שמסכם.
```

### כלל הסיום: **משפט אחד שמשאיר תחושה — לא סיכום יבש.**

## 8. מה הופך כתבה לאנושית — The Human Touch

### 5 טקטיקות שזוהו בפועל

1. **שבירת הרביעית (ביזפורטל)**:
   ```
   "ואגב זה קורה הרבה. לפני שאתם חותמים על ייעוץ משכנתא, תשקיעו שעה בחיפוש
   ברשת, תבדקו מול בנקים."
   ```
   → הכותב פונה ישירות לקורא, כאדם לאדם.

2. **ציטוט מיד עם שם תפקיד (כלכליסט)**:
   ```
   "המנכ"ל ליפ בו טאן, דמות אגדית בשוק השבבים שנכנס לתפקיד לפני כשנה"
   ```
   → אנשים אמיתיים עם ביוגרפיה קצרה.

3. **מספר + "בלבד" = הפתעה (כלכליסט)**:
   ```
   "פיצוי של 27 מיליון שקל בלבד" / "תשלום של 27 מיליון שקל בלבד"
   ```
   → "בלבד" מוסיף כעס/פליאה ללא מילים נוספות.

4. **תיאור חיים יומיומיים לפני הנתון (ביזפורטל)**:
   ```
   "ביתם היה קר בחורף. המקרר היה מלא בספרי קודש. כל אותו הזמן — 2 מיליון דולר
   בחשבון אמריקאי."
   ```
   → קונקרטי > מופשט.

5. **ייחוס יום ספציפי + שעה (כלכליסט)**:
   ```
   "ביום חמישי האחרון פירסמה אינטל..." / "היום (ו') הודיעו החברות..."
   ```
   → הקורא מרגיש שהוא קורא חדשות חמות.

## 9. פורמט הכתבה — Article Structure

```
[H1] כותרת: חדה, מתח, עד 10 מילים

[תת-כותרת] משפט הסבר: 20-30 מילים שמרחיבים את הכותרת
             (מפרט את "מה קרה" בלי לספר הכל)

[פסקת פתיחה] 2-3 משפטים — עובדה הכי מעניינת קודמת

[H2] כותרת משנה 1 — זווית ראשונה
[1-2 פסקאות]

[H2] כותרת משנה 2 — זווית שנייה
[1-2 פסקאות]

[H2] כותרת משנה 3 — (אופציונלי) זווית שלישית
[1-2 פסקאות]

[פסקת סיום] — תחושה/הבטה קדימה/עובדה אחרונה ממצה
```

סה"כ: 500–750 מילים. קצר וחד עדיף על ארוך ומשעמם.

## 10. ניתוח אתרי המקור — Site-Specific Patterns

### כלכליסט (calcalist.co.il)
- **טון**: אנליטי, מעמיק, לא עוצר בעובדות — מפרש
- **קטגוריות**: טק/הייטק, שוק הון, נדל"ן, כלכלה מקומית
- **כותרות**: מבנה דו-חלקי, ניסוח מחכים ("הברקס שלא התגלגל")
- **מאמרים ניכרים**: ניתוח מגמות, "מה זה אומר עבורך"

### ביזפורטל (bizportal.co.il)
- **טון**: עיתונאי יומיומי, מקרים אנושיים, נתונים גולמיים
- **קטגוריות**: בורסה/מסחר, נדל"ן ומשכנתאות, כלכלה ישראלית
- **כותרות**: שאלות, מחירים, מקרים ספציפיים עם שמות
- **מאמרים ניכרים**: פסיקות בית משפט, מחירי שכירות, נתוני בנייה
